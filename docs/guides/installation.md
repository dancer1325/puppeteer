# Installation

* | your project

  ```bash npm2yarn
  npm i puppeteer
  ```
  * AUTOMATICALLY downloads
    * [Chrome for Testing](https://developer.chrome.com/blog/chrome-for-testing/) | "$HOME/.cache/puppeteer"
    * if Puppeteer v21.6.0 -> `chrome-headless-shell` binary
    * if you want to configure it -> [here](configuration.md)

* | Puppeteer v1.7.0+,
  * published packages
    * [`puppeteer`](../../packages/puppeteer)
    * [`puppeteer-core`](../../packages/puppeteer-core)
